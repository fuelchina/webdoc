# 强制插件
通过 查找所有可用的 forc 插件`PATH`。

打印有关每个发现的插件的信息。

# 用法：
forc 插件 [选项]

# 选项：
`-d`，`--describe`

打印与每个列出的插件相关的详细描述

`-h`，`--help`

打印帮助信息

`-L`，`--log-level`<日志级别>

设置日志级别

`-p`，`--paths`

打印每个发现的插件的绝对路径

`-s`，`--silent`

使所有输出静音

`-v`，`--verbose`
使用详细输出

`-V`，`--version`

打印版本信息

示例：#列出所有插件 forc plugins

#列出所有插件及其路径 forc plugins --paths

#列出所有插件及其描述 forc plugins --describe

#列出所有插件及其路径和描述 forc plugins --paths --describe
