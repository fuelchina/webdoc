# Sway 语言基础知识

Sway 是一种专为 FuelVM 设计的编程语言。它是一种静态类型的编译语言，具有类型推断和特征。 Sway 旨在通过使用强大的静态分析和编译器反馈，使智能合约开发更安全、更高效。
